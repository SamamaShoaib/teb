# The Empowerment Bridge (TEB)
## Pakistan's Premier Ecosystem for Persons with Disabilities (PWD)

**TEB** is a project-level prototype designed to solve the economic and social isolation of the PWD community in Pakistan. It is a holistic platform that bridges the gap between talent, corporate inclusion, and financial support.

---

### 🚀 Technical Excellence
This prototype is built on a **Resilient Monorepo Architecture** designed for rapid scaling and high reliability.

- **Frontend**: React 19 + Vite + TailwindCSS 4 + Framer Motion (Cinematic UI).
- **Backend**: Express + Tsx + Drizzle ORM (Type-safe SQL).
- **Architecture**: Decoupled "Front/Back/Shared" structure ensuring a **Single Source of Truth** for API schemas and DB Types.
- **Safety**: Robust **Safe-Mode Authentication** and **Global Error Boundaries** ensure zero-crash performance during demos.
- **Resilience**: Integrated **Mock Mode** allows the platform to function perfectly even without an active internet connection or backend database.

---

### 🌟 Key Prototype Features
1. **Radical Accessibility**: Keyboard-first navigation and ARIA-compliant UI components.
2. **Impact Dashboard**: Real-time data visualization showing regional growth and placement metrics.
3. **AI Profile Builder**: Mock AI resume scanning for one-click user onboarding.
4. **Community Captains**: A mentorship layer connecting PWDs with local experts.
5. **Support & Funding**: A transparent grant application system with full admin oversight.
